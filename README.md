GO Mart - Premium E-Commerce Superstore

**GO Mart** is a modern, responsive, multi-page frontend e-commerce web application. Designed with a premium "Breadfast-inspired" Royal Blue and Orange aesthetic, GO Mart offers a sleek, user-friendly shopping experience for premium groceries and everyday items.

✨ Features

- **Multi-Page Architecture:** Fully interconnected static pages for an optimal user flow (`index`, `products`, `delivery`, `payment`, `contact`).
- **Dynamic Product Filtering:** A JavaScript-powered catalog featuring 20 premium, locally-hosted items categorized seamlessly into Bakery, Produce, Dairy, Pantry, and Beverages.
- **Premium UI/UX:** Built from scratch using raw CSS! Employs modern CSS Variables for easy theming, Glass morphism effects, floating input labels, and soft retail shadows.
- **Responsive Layout:** A fluid CSS grid system and a 100% width navigation ribbon ensure the site looks spectacular across all screen sizes.
- **Robust Local Asset Loading:** Bypasses CORS and strict browser blocking constraints by heavily relying on a bundled `assets/` folder serving high-quality product imagery directly.
- **Immersive Forms:** Beautifully styled shipping forms complete with customized dropdowns (including an Egypt governorates selector) and a secure payment checkout layout.

🛠️ Built With

- **HTML5:** Semantic, accessible architecture.
- **CSS3:** Flexbox, Grid Layouts, CSS custom properties, and responsive `@media` queries.
- **Vanilla JavaScript:** DOM Manipulation and event listeners for instantaneous category filtering.
- **Google Fonts:** Extensively utilizing 'Poppins' for a clean, modern typography.
